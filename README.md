# Student Grade Tracker

A Java application to manage and track student grades using **Core Java** concepts. This project demonstrates the use of **Collections**, **Swing GUI**, and basic **data visualization** techniques.

## Features

- Store student records and grades using **Java Collections**.
- Interactive **Swing GUI** for teachers and students to input and view grades.
- Calculate **GPA** automatically based on subject grades.
- Generate **progress reports** for individual students.
- Visualize performance with **charts** to track academic trends.

## Technologies Used

- Java SE (Core Java)
- Swing for GUI
- Java Collections (ArrayList, HashMap, etc.)
- Java I/O for storing/retrieving records
- Chart libraries (optional, e.g., JFreeChart) for performance visualization
